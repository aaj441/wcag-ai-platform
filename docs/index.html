API Quick Start

Authentication

All API requests require JWT authentication via Clerk:

curl -X GET https://api.wcag-ai-platform.com/api/scans \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json"

Create a Scan

curl -X POST https://api.wcag-ai-platform.com/api/scans/create \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com",
    "wcagLevel": "AA",
    "scanType": "full"
  }'
